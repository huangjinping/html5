# 接口文档

```
dependencies {
    implementation 'io.branch.sdk.android:library:5.+'
    implementation 'androidx.browser:browser:1.0.0'
     implementation  'com.google.android.gms:play-services-ads-identifier:17.1.0+'

}
```
#### 通用

###### 请求头部

| 标签          | 必填 | 说明                                             |
| ------------- | ---- | ------------------------------------------------ |
| client-id     | 是   | 当前产品号（子产品传appSsid）                    |
| token         | 否   | 登录令牌，登录时获取                             |
| userId        | 否   | 用户ID，登录时获取（子产品传curUserId）          |
| currentUserId | 否   | 当前用户ID，子产品传curUserId                    |
| channel       | 是   | 频道，默认传`googleplay`                         |
| versionName   | 是   | 版本名                                           |
| versionCode   | 是   | 版本号                                           |
| device-id     | 是   | 设备ID                                           |
| imei          | 是   | 设备识别码, deviceId与imei保持一致               |
| mulFlag       | 是   | 多产品标识，默认传`1`                            |
| v-flag        | 否   | 登录标识，默认传`空`（不需要登录的接口传`true`） |
| deviceId      | 是   | 设备ID, deviceId与imei保持一致                   |

###### 请求参数`Form-data`

| 参数名          | 类型     | 必填 | 说明                               |
| --------------- | -------- | ---- | ---------------------------------- |
| appssid         | [string] | 是   | 当前产品号（子产品传appSsid）      |
| userId          | [int]    | 否   | 登录时获取（子产品传curUserId）    |
| lbs             | [string] | 是   | 经纬度，无法定位时传`0,0`          |
| language        | [string] | 是   | 语言，默认传`es`                   |
| versionName     | [string] | 是   | 版本名称                           |
| versionCode     | [string] | 是   | 版本号                             |
| deviceId        | [string] | 是   | 设备ID, deviceId与imei保持一致     |
| imei            | [string] | 是   | 设备识别码, deviceId与imei保持一致 |
| channel         | [string] | 是   | 频道，默认传`googleplay`           |
| systemMode      | [string] | 是   | 设备型号                           |
| googleMobileNo  | [string] | 否   | 登录手机号                         |
| googleGaid      | [string] | 是   | 谷歌广告id                         |
| googleUserAgent | [string] | 是   | 用户代理                           |

###### 返回参数

| 参数名 | 类型     | 必含 | 说明                                                  |
| ------ | -------- | ---- | ----------------------------------------------------- |
| code   | [int]    | 是   | 1000: 成功<br/>-1001: token过期<br/>-1000(其他): 失败 |
| msg    | [string] | 是   | 错误信息                                              |
| data   | [object] | 否   | 响应数据                                              |

###### *code枚举*

| 标签  | 说明               |
| ----- | ------------------ |
| 1000  | 成功               |
| -1001 | 需要登录或令牌失效 |
| -1000 | 失败               |
| 其他  | 其他未知错误       |

###### 返回示例

```json
{"msg":"system exception","code":-1000}
```
